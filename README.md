# Digital Blackjack Game

A simple digital card game built with Svelte, TypeScript, and Tailwind CSS. It features glassmorphic playing cards, 3D tactile buttons, and custom dealing animations.

---

## Features

* **Glassmorphic Cards:** High-contrast card layouts backed by custom 3D card backs.
* **Tactile Controls:** Interactive 3D buttons that physically depress upon clicking.
* **Sequential Dealing:** Automated dealer routines that alternate card distribution step-by-step.
* **Input Validation:** Integrated rate-limiting and maximum card boundaries to ensure stable state changes.

## Getting Started

### Prerequisites

Verify that Node.js is installed locally on your system.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/thapasubham/blackjack-game

2. **Navigate to the root directory:**
  ```
cd blackjack-game
```

4. **Install node modules:**
 ```
npm install
```
## Running the project 
```
npm run dev
```

- Framework: Svelte
